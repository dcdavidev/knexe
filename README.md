## Workspace Dependencies

### Required

- **nx**: monorepo/workspace manager.
- **commitizen**: conventional commits cli.
- **prettier**: formatter.
- **eslint**: linter.

```bash
npm install --global
nx \
  commitizen \
  prettier \
  eslint
```
